## 🏗️ 技术架构

### 前端技术栈
- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS 3
- **语言**: TypeScript
- **状态管理**: Zustand
- **表单处理**: React Hook Form + Zod
- **UI组件**: 自定义组件库
- **动画**: Framer Motion
- **图标**: Heroicons

### 后端技术栈
- **框架**: FastAPI
- **数据库**: PostgreSQL
- **缓存**: Redis
- **文件存储**: MinIO (S3兼容)
- **向量数据库**: ChromaDB
- **认证**: JWT
- **API文档**: Swagger/OpenAPI

### 开发工具
- **代码规范**: ESLint + Prettier
- **测试**: Jest + React Testing Library + Playwright
- **部署**: Docker + CI/CD
- **监控**: Sentry

## 💡 技术决策记录

### 状态管理选择 - Zustand
**原因**: 
- 轻量级，bundle size小
- TypeScript支持优秀
- 中间件生态丰富
- 学习曲线平缓

### UI组件策略 - 自建组件库
**原因**:
- 完全控制设计系统
- 避免第三方依赖过重
- 更好的定制性
- 学习和掌握组件设计

### 文件存储 - MinIO
**原因**:
- S3兼容API
- 私有部署支持
- 性能优秀
- 企业级特性
